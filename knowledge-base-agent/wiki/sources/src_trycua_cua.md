# CUA — Computer-Use Agents Infrastructure

**Tác giả:** trycua
**Nội dung chính:** Hạ tầng mã nguồn mở cho các AI agent có khả năng điều khiển máy tính (Computer-Use). Cung cấp sandbox, SDKs và benchmarks để huấn luyện và đánh giá agent trên nhiều hệ điều hành (macOS, Linux, Windows).
