## 4 API Products

| Product | Chức năng |
|---|---|
| **Search API** | Fast, structured web search — real browser-rendered, JSON output |
| **Fetch API** | Any URL → clean content (markdown, JSON, HTML) — rendered in real browser |
| **Agent API** | Multi-step web automation — navigate, fill forms, authenticate |
| **Browser API** | Stealth Chromium sessions — bypass anti-bot, sub-250ms cold starts |
