# GPT-5.5 Instant — Smarter, Clearer, More Personalized
**Link:** https://openai.com/index/gpt-5-5-instant  
**Release date:** May 5, 2026
---
## Tóm tắt
OpenAI ra mắt **GPT-5.5 Instant** — model mặc định mới cho ChatGPT, thay thế GPT-5.3 Instant. Tập trung vào 3 cải tiến chính: thông minh hơn, cô đọng hơn, cá nhân hóa tốt hơn.
---
## 1. Smarter & More Accurate
| Metric | Improvement |
|---|---|
| Hallucination reduction | **52.5% fewer** hallucinated claims (medicine, law, finance) |
| Inaccurate claims | **37.3% fewer** on challenging conversations |
| STEM questions | Improved |
| Image analysis | Better photo/image understanding |
| Web search | Better at deciding when to use |
**Example (algebra):** GPT-5.5 bắt lỗi tốt hơn, recover từ initial mistake, identify actual algebra error.
---
## 2. Clearer & More Concise
- **30.2% fewer words**
- **29.2% fewer lines**
- Ít verbosity, overformatting
- Ít unnecessary follow-up questions
- Ít gratuitous emojis
- Giữ warmth và personality
---
## 3. More Personalized
- Sử dụng context từ past chats, files, Gmail (nếu connected)
- Intelligent decision khi nào nên personalize
- Faster search past conversations
- **Memory sources:** Visibility over what context được dùng để personalize
---
## Availability
- Rolling out now — default model cho all ChatGPT users
- API: `chat-latest`
- GPT-5.3 Instant: available 3 months for paid users → then retired
- Enhanced personalization: Plus/Pro users first → expand to others
