# GPT-5.5 Instant — Smarter, Clearer, More Personalized

**Link:** https://openai.com/index/gpt-5-5-instant  
**Release date:** May 5, 2026

---

## Mô tả

OpenAI ra mắt **GPT-5.5 Instant** — model mặc định mới cho ChatGPT, tập trung vào 3 cải tiến: thông minh hơn, cô đọng hơn, cá nhân hóa tốt hơn.

---

## 3 Improvements

| Aspect | Improvement |
|---|---|
| **Smarter** | 52.5% fewer hallucinated claims, 37.3% fewer inaccurate claims |
| **Clearer** | 30.2% fewer words, 29.2% fewer lines |
| **More Personalized** | Context từ past chats, files, Gmail |

- Rolling out now — default model cho all ChatGPT users
- API: `chat-latest`
