# Closed Loop System

## Definition

Closed Loop System (vòng lặp khép kín) là hệ thống trong đó agent/output có thể tự kiểm tra, tự validate và tự iterate mà không cần sự can thiệp liên tục của con người. Đây là cách vượt qua giới hạn ~3 agents song song mà con người có thể quản lý.

## Key ideas

- **Close the loop end-to-end:** Tests, queries, sandboxes, browser tools, type checks, real API responses
- **Agent thấy output của chính nó** và iterate without human
- **Work runs underneath cognitive window** vì không cần trong đó
- **AI có thể handle "fuzzy" verifications** với predefined context đúng
