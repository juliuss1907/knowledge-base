# Last30days Skill

## Definition

Skill của Hermes thực hiện 30-day multi-source alpha scan từ Reddit, X, YouTube, GitHub — aggregate insights từ multiple sources để identify trends và opportunities.

## Key ideas

- **30-day scan**: Time window cố định để analyze
- **Multi-source**: Reddit, X, YouTube, GitHub
- **Alpha identification**: Tìm kiếm early signals và opportunities
- **Trend detection**: Nhận diện emerging trends
- **Automated aggregation**: Tự động collect và synthesize data từ nhiều nguồn
- **Complement other skills**: Kết hợp với Hindsight và Browser Use
