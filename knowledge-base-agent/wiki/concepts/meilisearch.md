# Meilisearch

## Tóm tắt

Search engine RESTful API, mã nguồn mở, hỗ trợ AI-powered hybrid search. Tốc độ tìm kiếm dưới 50ms. Dễ tích hợp vào app/website chỉ trong vài dòng code.

## Tính năng chính

**Tìm kiếm:**
- Hybrid search — semantic + full-text
- Search-as-you-type: dưới 50ms
- Typo tolerance, synonyms, filtering, faceted search
- Geosearch, multi-language support

**AI:**
- Conversational search
- Personalization
- Vector search (semantic search)
- Tích hợp LangChain và MCP

**Triển khai:**
- RESTful API, nhiều SDK (Python, JS, Go, Ruby...)
- Cloud option có sẵn
- MIT license cho core, BSL 1.1 cho enterprise features
