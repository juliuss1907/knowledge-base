# AI Agent Setup Mistakes

## Definition

Các lỗi phổ biến khi setup AI agents và cách fix — từ Windows setup issues đến subscription choices và cost optimization.

## Key ideas

### Mistake 1: Windows Setup

**Vấn đề:** Hermes works great với Mac. Cài trên Windows gây lỗi commands, cron jobs, editing configs.

**Fix:** Re-install Hermes through **WSL2 via Ubuntu** → works great

### Mistake 2: Claude Max Subscription

**Vấn đề:**
- Subscribe Claude Max, waste tokens
- Anthropic banned agents 1-2 tuần trước
- Switch to Openrouter → burn $$$ fixing setup bugs

**Fix:** Subscribe **Opencode Go $5/tháng đầu**
- Saves tokens
- Kimi k2.5 và GLM5.1 cho long-running tasks
- MiniMax 2.7 cho conversations

### Key Insight

> "An AI without the right setup is like a Ferrari without wheels — incredible engine, but it's not going anywhere."
> "An agent without the right direction is like a fighter jet without a pilot — it can fly sky high, but burning fuel in a straight line until it dies."
