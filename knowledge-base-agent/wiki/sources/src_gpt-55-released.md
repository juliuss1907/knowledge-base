# GPT-5.5 Released: First Fully Retrained Base Model Since GPT-4.5, 1M Context, $5/$30 Pricing

Thông tin ra mắt GPT-5.5 của OpenAI, đánh dấu bước chuyển sang mô hình tập trung vào Agent.

## Key Points
- **Đặc điểm**: Là mô hình base được huấn luyện lại toàn bộ (fully retrained), hỗ trợ context window 1M tokens.
- **Định hướng Agent**: Được tối ưu để thực hiện chuỗi hành động, sử dụng công cụ, tự kiểm tra kết quả và tiếp tục cho đến khi hoàn thành task mà không cần nhắc nhở từ con người ở mỗi bước.
- **Phiên bản & Giá**:
    - **Thinking**: $5/$30 per million tokens.
    - **Pro**: $30/$180 per million tokens.
- **Hiệu suất**:
    - Vượt trội trong Terminal-Bench 2.0 và GDPval (knowledge work).
    - Vẫn thua Opus 4.7 ở SWE-Bench Pro (giải quyết codebase).
    - Tỷ lệ hallucination cao (86%), cao hơn đáng kể so với Opus 4.7 (36%).
- **Chi phí**: Giá tăng gấp đôi so với GPT-5.4, nhưng OpenAI cho rằng mô hình tiêu tốn ít token hơn (~40%) cho cùng một tác vụ.
