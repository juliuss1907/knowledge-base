# Reflect Skill with Hindsight

## Definition

Skill "Reflect" của Hermes kết hợp với Hindsight (external memory provider) để synthesize thông tin quá khứ, kết nối relationships, detect patterns, cung cấp actionable analysis và top daily insights.

## Key ideas

- **Reflect:** Synthesize thông tin quá khứ
- **External Memory Provider:** Hindsight — #1 recall accuracy, especially long-horizon, multi-session, large-scale
- **Pattern detection:** Nhận diện patterns từ historical data
- **Actionable analysis:** Cung cấp insights có thể hành động
- **Top 5 Daily insights:** Highlight signals và giải thích tại sao quan trọng

**Cảnh báo token cost:** Đừng link Hindsight to OpenRouter + Claude Sonnet 4.6 → Burn $50+ tokens trong 1 ngày
