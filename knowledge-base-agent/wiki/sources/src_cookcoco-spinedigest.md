# I built an open-source tool to distill books into knowledge graphs

**Tác giả:** Cookcoco
**Nội dung chính:** Giới thiệu SpineDigest, một công cụ CLI mã nguồn mở dùng để biến sách thành đồ thị kiến thức (knowledge graph) thay vì tóm tắt phẳng. Quy trình gồm: trích xuất chunk -> xây dựng đồ thị -> tóm tắt đối kháng. Có ứng dụng Inkora để trực quan hóa kết quả.
