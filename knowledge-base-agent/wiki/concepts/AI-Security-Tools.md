# Công cụ Bảo mật AI (AI Security Tools)

## Mô tả
Các mô hình AI chuyên biệt được thiết kế cho mục đích bảo mật, bao gồm kiểm thử xâm nhập (penetration testing), tìm lỗ hổng và phân tích mã độc. Do tính chất nguy hiểm nếu bị lạm dụng, các công cụ này thường bị giới hạn truy cập.

## Liên quan
- An toàn AI (AI Safety)
- Cyber Security
- GPT-5.5 Cyber / Anthropic Mythos
