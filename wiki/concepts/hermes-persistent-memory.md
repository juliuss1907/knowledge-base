# Hermes Persistent Memory

## Definition

Khả năng của Hermes duy trì thông tin và context giữa các sessions, remember how you do things và how you operate — khác biệt quan trọng so với Claude/Codex bị mất context khi xóa session.

## Key ideas

- **Khác biệt với Claude**: Claude quên bạn giữa các session, xóa session = mất hết context
- **Remember operations**: Nhớ cách bạn làm việc, preferences, patterns
- **No token waste**: Không cần re-explain context mỗi lần
- **Perfect for analyst**: Biết holdings, portcos, thesis → identify opportunities
- **Self-learning**: Learn everyday để deliver better outputs
- **Combine skills**: Kết hợp nhiều skills với nhau dựa trên memory
