## Summary

Từ góc nhìn của người có 8 năm kinh nghiệm PE/VC/IB, tác giả chia sẻ 3 điều học được sau 3 tuần dùng Hermes như personal analyst: Windows setup qua WSL2, dùng Opencode Go thay Claude Max, và các skills/tools hữu ích. Chi phí chỉ $5-10/tháng với workflow mới: check Discord → Hermes → feedback → improve.

## Key points

- **Persistent memory**: Hermes nhớ cách bạn làm việc giữa các session, khác với Claude quên giữa sessions
- **Windows Setup Mistake**: Cài trên Windows gây lỗi → Fix bằng WSL2 via Ubuntu
- **Claude Max Mistake**: Subscribe Claude Max waste tokens, Anthropic banned agents → Fix bằng Opencode Go $5/tháng
- **Use cases**: Track macro/geopolitics, X bookmarks summarize, portfolio report, learn from smart people
- **Hindsight skill**: Memory provider — "reflect" on experiences, analyze relationships & patterns
- **Last30days skill**: 30-day multi-source alpha scan từ Reddit, X, YouTube, GitHub
- **Phase plan**: Phase 1 optimize workflows ($10-20/tháng), Phase 2 give wallet cho trading experiments
- **Cost**: Chỉ $5-10/tháng sau optimization
