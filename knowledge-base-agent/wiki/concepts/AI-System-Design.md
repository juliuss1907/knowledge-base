# Thiết kế hệ thống AI cá nhân (Personal AI System Design)

## Mô tả
Một kiến trúc hệ thống AI di động và linh hoạt, tập trung vào việc làm chủ dữ liệu, giảm sự phụ thuộc vào nhà cung cấp model và tối ưu hóa hiệu suất thông qua các workflow chuẩn hóa.

## Liên quan
- Tự động hóa (Automation)
- Quản trị tri thức (Knowledge Management)
- MCP (Model Context Protocol)
- Framework SIGNAL
