# Model Context Protocol (MCP) in Crypto Trading

## Mô tả
Sử dụng MCP để kết nối Claude với dữ liệu real-time từ các nguồn crypto, biến AI thành một trợ lý nghiên cứu và giao dịch chuyên sâu.

## Chi tiết
- **MCP là gì:** Giao thức cho phép AI truy cập dữ liệu bên ngoài training set, cung cấp ngữ cảnh thời gian thực.
- **Ứng dụng trong Crypto:**
    - **Technical Analysis:** TradingView MCP.
    - **On-chain Data:** Dune, CoinGecko, DeFi Portfolio MCPs.
    - **Sentiment & News:** LunarCrush, CryptoPanic MCPs.
    - **Execution:** GOAT SDK cho swap và transfer.
- **Cách tiếp cận:** Có thể dùng qua Claude Projects (không code) hoặc xây dựng Dashboard tùy chỉnh (Vibe-coded).
