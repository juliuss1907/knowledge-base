# How To Turn Claude Into A Crypto Trading Beast (10 MCPs)

## Tóm tắt

Kết nối Claude với crypto qua MCP (Model Context Protocol) — biến Claude thành công cụ nghiên cứu và giao dịch crypto mạnh nhất.

## MCP là gì

MCP = Model Context Protocol. Là cầu nối giữa Claude và các nguồn dữ liệu bên ngoài. Không có MCP, Claude chỉ có training data cũ. Có MCP, Claude truy cập dữ liệu real-time.

## 10 Crypto MCPs được recommend

| # | MCP | Công dụng |
|---|---|---|
| 1 | **TradingView** | Chart real-time, alerts, technical analysis |
| 2 | **Perplexity** | Tìm kiếm live web cho research |
| 3 | **Dune** | On-chain analytics, SQL queries |
| 4 | **GOAT SDK** | On-chain execution, swaps, token transfers |
| 5 | **altFINS** | 300+ data points/coin, RSI, MACD, trading signals |
| 6 | **CoinGecko** | Price feeds, token analytics, DEX data |
| 7 | **DeFi Portfolio** | Đọc positions Aave, Compound, Morpho, Uniswap V3 |
| 8 | **CryptoPanic** | Tin tức real-time, regulatory updates |
| 9 | **CoinMarketCap** | Market data, rankings, Fear & Greed index |
| 10 | **LunarCrush** | Social sentiment, engagement scores |

## Cách cài đặt

1. Tải Claude Desktop App + Node.js
2. Mở Claude → Customize → Connectors (1 MCP có sẵn)
3. Hoặc gõ `Claude mcp add [server-url]` trong Claude Code

## Ví dụ prompts

**LunarCrush:**
- "Top 10 coins by social sentiment right now?"
- "Coins có spike social engagement nhưng price chưa react?"

**CoinMarketCap:**
- "Top 20 cryptocurrencies by market cap"
- "Fear and Greed index và top macro events hôm nay"

**DeFi Portfolio:**
- "Full breakdown of all DeFi positions across protocols"
- "Uniswap V3 positions nào đang out of range?"

**altFINS:**
- "Coins vừa cross above SMA200 với volume tăng"
- "Mid-cap altcoins oversold nhưng still in uptrend"

## Hai cách sử dụng

### 1. Claude Projects (không cần code)
Tạo "Crypto Research Hub" project, paste system prompt để activate tất cả MCPs.

### 2. Vibe-Coded Dashboard
Dùng prompt để code dashboard HTML visualize tất cả data từ các MCPs.

## Lưu ý

- GOAT SDK và DeFi Portfolio cần wallet access → dùng test wallet trước
