# User Agent cho AI

## Mô tả
User Agent cho AI là một lớp đại diện (representative layer) nằm giữa AI agent và môi trường bên ngoài, đóng vai trò bảo vệ lợi ích của người dùng tương tự như cách trình duyệt web (Web Browser) hoạt động. Thay vì cho phép AI agent truy cập dữ liệu và thực hiện hành động không kiểm soát, User Agent thiết lập các ràng buộc, tiêu chuẩn quyền hạn và cơ chế sandboxing để đảm bảo quyền riêng tư và an toàn.
