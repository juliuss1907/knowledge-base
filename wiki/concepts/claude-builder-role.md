# Claude Builder Role

## Definition

Claude (và các AI code assistant như Codex) đóng vai trò **Builder** — chuyên xây dựng dashboard, websites, slides, excel, UI/UX changes — các công việc building one-time. Đây là vai trò bổ sung cho Operator của các agent như Hermes.

## Key ideas

- **Build nhanh gấp 10x**: Xây dashboard, website, UI nhanh hơn Hermes đáng kể
- **Built-in features tốt hơn**: Có các features giúp non-technical navigate dễ hơn
- **Aesthetics tốt hơn**: UI/UX đẹp hơn so với output của Hermes
- **Không phù hợp recurring tasks**: Thiếu persistent memory và self-learning loop
- **Kết hợp với Operator**: Build dashboard bằng Claude, Operator chạy analyze định kỳ
