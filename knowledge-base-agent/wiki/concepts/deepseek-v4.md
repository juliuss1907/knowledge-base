# DeepSeek V4

## Mô tả
DeepSeek V4 là một dòng mô hình ngôn ngữ lớn (LLM) mã nguồn mở (giấy phép Apache 2.0) được phát triển bởi DeepSeek. Mô hình sử dụng kiến trúc Mixture of Experts (MoE) với hai biến thể chính là Pro (quy mô 1.6T parameters) và Flash (quy mô 284B parameters). DeepSeek V4 nổi bật với khả năng xử lý ngữ cảnh cực lớn (lên đến 1 triệu tokens), hiệu suất lập trình vượt trội (đặc biệt trên Codeforces) và mức giá API cực kỳ rẻ so với các mô hình đóng như GPT-5.5 hay Claude Opus.
