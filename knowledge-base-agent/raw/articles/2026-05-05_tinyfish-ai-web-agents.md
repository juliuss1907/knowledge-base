## Tóm tắt
TinyFish cung cấp **enterprise infrastructure** cho AI web agents — nền tảng all-in-one cho search, fetch, browser sessions và autonomous web workflows.
---
## 4 API Products
| Product | Chức năng |
|---|---|
| **Search API** | Fast, structured web search — real browser-rendered, JSON output, never cached |
| **Fetch API** | Any URL → clean content (markdown, JSON, HTML) — rendered in real browser |
| **Agent API** | Multi-step web automation — navigate, fill forms, authenticate, extract structured results |
| **Browser API** | Stealth Chromium sessions — bypass anti-bot, sub-250ms cold starts |
→ **One API key** cho tất cả products.
