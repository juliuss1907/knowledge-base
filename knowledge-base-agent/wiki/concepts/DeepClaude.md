# DeepClaude

## Mô tả
DeepClaude là một công cụ cho phép vận hành agent loop tự trị của Claude Code CLI nhưng thay thế model Anthropic đắt đỏ bằng các model giá rẻ hơn như DeepSeek V4 Pro. Nó hoạt động như một proxy đánh chặn các yêu cầu API từ Claude Code và điều hướng chúng đến các provider khác (DeepSeek, OpenRouter, Fireworks AI) mà vẫn giữ nguyên trải nghiệm người dùng (UX) và khả năng thao tác file/bash/git.
