# Source: Ternary Bonsai — 1.58-bit Language Models

## Tóm tắt
Giới thiệu Ternary Bonsai, model 1.58-bit (-1, 0, +1) từ PrismML giúp giảm footprint bộ nhớ và tăng tốc độ xử lý.

## Key Points
- **Hiệu năng:** Bản 8B đạt score 75.5, tiệm cận Qwen3 8B nhưng nhỏ hơn 9-10 lần.
- **Footprint:** Ternary Bonsai 8B chỉ tốn 1.75 GB RAM.
- **Tốc độ:** 82 toks/sec trên M4 Pro, 27 toks/sec trên iPhone 17 Pro Max.
- **Tiết kiệm:** Hiệu suất năng lượng tốt hơn 3-4 lần so với model 16-bit.
