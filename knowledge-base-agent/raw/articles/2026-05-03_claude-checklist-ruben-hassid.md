# Claude Checklist — Ruben Hassid

**Nguồn:** Infographic by Ruben Hassid  
**Link:** https://how-to-ai.guide  
**Tác giả:** Ruben Hassid

---

## Tóm tắt

Infographic hướng dẫn toàn diện cách tối ưu Claude AI với 9 danh mục checklist.

---

## 1. Setup

- Trả Pro $20/tháng
- Tải desktop app
- Chọn Opus 4.7 với Extended Thinking
- Bật Computer Use trong Settings
- Kết nối Dispatch vớI điện thoạI
- Cài Claude trong Excel từ add-in store
- Cài Wispr Flow.ai cho voice input
- Cài Obsidian và mở folder Cowork

---

## 2. Context Folder

Tạo folder "Claude Cowork" với 3 subfolders:
- **ABOUT ME** — Thông tin cá nhân
- **OUTPUTS** — Kết quả đầu ra
- **TEMPLATES** — Mẫu

Checklist:
- Viết file about-me.md (dưới 2,000 từ)
- Viết anti-ai-writing-style.md vớI các từ cấm
- Dán global instructions trong Settings
- Yêu cầu Cowork đọc ABOUT ME trước mọi task
- Yêu cầu Cowork KHÔNG đọc OUTPUTS hoặc TEMPLATES
- Lưu deliverables vào OUTPUTS trong project subfolder

---

## 3. Prompting

- Bắt đầu vớI điều bạn muốn và thế nào là thành công
- Thêm "use AskUserQuestion before you start" vào mọi prompt
- Cho một ví dụ output tốt thay vì mô tả
- Nói prompt bằng Wispr Flow khi context quan trọng
- Sửa messages thay vì gửi follow-ups
- Nhóm 3 tasks vào một message
- Nói cho Claude làm gì, không phảI tránh gì
- Mở chat mớI khi topic thay đổI

---

## 4. Connectors

Kết nối:
- Gmail (soạn/đọc)
- Gamma (pitch decks)
- Google Drive (truy cập tài liệu)
- Notion (workspace pages)
- Granola (transcripts)
- Calendar (lên lịch)
- GitHub (code)

**Tip:** Tắt connectors không cần thiết cho task cụ thể

---

## 5. Skills

- Hỏi "skill-creator" trong Cowork để được hỗ trợ
- Trả lờI interview questions cụ thể
- Chạy evaluation mà Claude tạo ra trước khi save
- Lưu skills trong Settings dưới Capabilities
- Test skills vớI 5 cách diễn đạt khác nhau
- Thêm dòng "do NOT use for" vào skill descriptions
- Build skills từ past conversations bằng right-click
- Browse Anthropic's plugin library cho pre-built skills

---

## 6. Projects

- Tạo một Project cho mỗi recurring deliverable
- Import old browser Projects vào Cowork
- Upload một "gold-standard" example cho mỗi Project
- Viết Project instructions dưới 8 dòng
- Setup scheduled tasks cho weekly work
- Giữ desktop app chạy cho scheduled tasks
- Dùng "read everything in this folder" làm first prompt
- Lưu winning outputs vào TEMPLATES folder

---

## 7. Design & Code

- Dùng claude.ai/design cho landing pages
- Upload DESIGN.md brand file
- Dùng "Tweaks" cho layout, "Edit" cho thay đổI pixel
- Dùng Claude Code cho full websites
- Kết nốI free GitHub account cho live deployment
- Cài VS Code vớI "Skip Permissions" cho vibecoding
- Tạo CLAUDE.md file trong mọI code project

---

## 8. Research & Search

- Bật Web Search trong prompt bar
- Dùng Research mode cho deep reports
- Yêu cầu Claude thực hiện 5 searches đa dạng và identify gaps
- Lưu findings vào research-brief.md
- Dùng Grok cho real-time news, ChatGPT cho deep research
- Yêu cầu Claude flag conflicting sources
- Ưu tiên nguồn 2025-2026 trong prompts

---

## 9. Token Economy

- Plan in chat trước, build in Cowork second
- Convert PDFs và screenshots sang markdown trước khi upload
- Restart conversations mỗi 20 messages
- Switch sang Sonnet cho grammar và câu trả lờI ngắn
- Tắt Extended Thinking cho simple tasks
- Chọn zero folders khi không cần files
- Chạy scheduled tasks cho recurring work
- Spread sessions across morning, afternoon, và evening
