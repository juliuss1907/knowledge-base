# Before GitHub

**Tác giả:** Armin Ronacher
**Nội dung chính:** Suy ngẫm về sự chuyển dịch của Open Source từ thế giới nhỏ, dựa trên uy tín cá nhân và hạ tầng tự vận hành sang kỷ nguyên trung tâm hóa của GitHub. Tác giả phân tích những lợi ích (dễ dàng đóng góp, lưu trữ lịch sử) và tác hại (vấn đề micro-dependency, sự phụ thuộc vào một nền tảng duy nhất) và đề xuất xây dựng một kho lưu trữ (Archive) công cộng, phi lợi nhuận cho mã nguồn mở.
