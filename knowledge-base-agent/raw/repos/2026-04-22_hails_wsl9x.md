# wsl9x — Windows 9x Subsystem for Linux

## Tóm tắt

Chạy Linux kernel (6.19) bên trong Windows 9x kernel. Linux và Windows 9x chạy song song trên cùng một máy, không cần reboot.

## Cách hoạt động

WSL9x cho phép Linux kernel chạy "coopertively" bên trong Windows 9x kernel:
- Tận dụng capabilities của cả hai OS cùng lúc
- Có paging, memory protection, pre-emptive scheduling
- Chạy các ứng dụng side by side — không reboot

## Yêu cầu để build

- Cross toolchain targeting `i386-linux-musl` (dùng musl-cross-make)
- Open Watcom v2 toolchain (build Windows components)
- Linux kernel patched từ branch `win9x-um-6.19`
- Hard drive image `hdd.base.img` với Windows 9x pre-installed

## Cách build & chạy

```bash
# Build patched Linux kernel
make defconfig ARCH=um SUBARCH=i386 KBUILD_DEFCONFIG=win9x
make vmlinux ARCH=um SUBARCH=i386 -j $(nproc)

# Set env vars (WATCOM, LINUX)
# Tạo hdd.img với Windows 9x

# Chạy
make        # produce hdd.img với WSL9x ready
wsl         # Mở pty tại MS-DOS prompt
```
