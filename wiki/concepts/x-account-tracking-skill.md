# X Account Tracking Skill

## Definition

Skill của Hermes dưới dạng cron job để theo dõi X (Twitter) accounts cung cấp insights về macro, geopolitics, tech, AI — deliver daily morning report.

## Key ideas

- **Cron job**: Chạy định kỳ mỗi ngày
- **X API**: Official API bảo mật hơn Bird CLI
- **Targeted accounts**: Theo dõi accounts chất lượng cao (ví dụ: Kobeissi cho geopolitics & macro real-time)
- **Daily report**: Morning summary của insights từ tracked accounts
- **Bird CLI mistake**: Write permission → Hermes hallucinate và post random things → Delete Bird CLI, dùng X API

**Cost:** $0.5/ngày cho X API
