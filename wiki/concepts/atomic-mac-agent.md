# Atomic (macOS AI Agent)

## Definition

Atomic là native macOS AI assistant — không phải browser tab, không phải CLI wrapper, không phải "ChatGPT với nút bấm". Đây là autonomous agent với tay, mắt, bộ nhớ, và workspace thực, cung cấp one-click setup cho Hermes Agent.

## Key ideas

- **Native macOS app:** Tích hợp sâu vào hệ thống, không phải web wrapper
- **100+ integrations:** Kết nối với nhiều tools và services
- **Pre-installed skills library:** Thư viện skills sẵn có lớn
- **Persistent memory:** Agent nhớ context và lịch sử qua các session
- **Multi-model support:** Claude, ChatGPT, Gemini, và local models (Gemma, Qwen, GLM)
- **Cloud option:** Có thể chạy trên cloud thay vì local
- **Sử dụng Hermes:** Chọn Hermes agent từ main page, connect model API, bắt đầu build
