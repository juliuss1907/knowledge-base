# Soul.md Configuration

## Definition

File configuration quan trọng nhất trong Hermes — định nghĩa "Who the agent is" (AI identity): mục tiêu sử dụng, vai trò, cách nói chuyện, operating constraints, giá trị, và kiến thức về user. ROI cao nhất trong 3 layers.

## Key ideas

- **"Who the agent is":** Định nghĩa identity của AI
- **Mục tiêu sử dụng:** Tại sao bạn dùng agent
- **Vai trò:** analyst, writer, coder, assistant
- **Cách nói chuyện:** professional, casual (có thể feed articles để agent nói như bạn)
- **Operating constraints:** Ví dụ: unit economics lens (10+ năm PE/VC/IB)
- **Investment theses, portfolio positions, risk tolerance**

**Thời gian:** 2-3 giờ viết, sửa 5+ lần

**Không bao giờ "hoàn thành":** Soul sửa 5-6 lần, cải thiện qua reflection
