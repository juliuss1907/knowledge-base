## Summary

Workflow tối ưu để sử dụng Hermes như analyst cá nhân, dựa trên kinh nghiệm #1 trên OpenRouter của tác giả. Ba lớp của agent hữu ích: Identity (Soul.md — ROI cao nhất), Knowledge (User.md + Memory), và Tools (Config + Skills). Model config recommendations, skill management best practices, và memory setup với Hindsight.

## Key points

- **3 Layers của agent:** (1) Identity/Soul.md — ROI cao nhất, (2) Knowledge/User.md — compound mỗi session, (3) Tools/Config — table stakes
- **Soul.md:** 2-3 giờ viết, sửa 5+ lần — quyết định mọi thứ downstream (personality, voice, constraints, values)
- **Model recommendations:** MiniMax 2.7 (basic), Kimi k2.6/GLM5.1 (complex), DeepSeek v4 (discount 75% tháng 5)
- **Skill auto-creation:** Agent tự tạo skill khi thấy task lặp lại
- **Browser Harness:** Flexible nhất nhưng chậm hơn direct tools
- **Hindsight:** External memory provider cho knowledge ingestion
- **Health check:** Delegate task cho sub-agent để check định kỳ
