# User.md Configuration

## Definition

File configuration layer 2 của Hermes — định nghĩa "What the agent knows about you": portfolio, thesis, sign-offs, lỗi lầm trước đây. Compound mỗi session, càng sửa càng hay.

## Key ideas

- **"What the agent knows":** Knowledge về user
- **Portfolio:** Holdings, positions, allocations
- **Thesis:** Investment theses, strategies
- **Sign-offs:** Quyết định đã thực hiện
- **Mistakes:** Lỗi lầm trước đây để learn from
- **Preferences:** Cách làm việc, risk tolerance

**Thời gian:** 30 phút bản nháp đầu, cập nhật hàng ngày

**Compound effect:** Càng sửa càng chính xác, càng useful
