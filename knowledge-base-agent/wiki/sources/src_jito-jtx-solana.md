# Jito — Why We're Building JTX

**Author:** @jito_sol  
**Link:** https://x.com/jito_sol/status/2051696304456184293

---

## Mô tả

Jito công bố **JTX** — trading app mới cho Solana, built cho "serious traders" không phải tourists. Hyperliquid lesson: blockchains ready for traditional finance.

---

## Vấn đề hiện tại

- Solana có infrastructure tốt nhất crypto (400ms settlement)
- Nhưng ecosystem xây dựng piecemeal: swap router, charting, portfolio tracker
- Traders phải dùng **5-8 different tools**

---

## JTX Features

| Feature | Description |
|---|---|
| One unified surface | Charts + Execution + Portfolio + Capital |
| Maker limit orders | Fill dependably, display on chart |
| Self-custody default | Your wallet is your wallet |
| Built for competence | Not for tourists |
