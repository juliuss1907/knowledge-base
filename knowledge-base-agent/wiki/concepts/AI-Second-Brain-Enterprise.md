# AI Second Brain cho Enterprise — Meta Case Study

## Mô tả

Meta xây dựng **AI Second Brain** từ experiment trong analytics org đến **63,000+ users** across mọi function chỉ trong 3 tháng. Viral adoption không có top-down directive.

## 4 Components chính

1. **PARA Workspace** — Projects, Areas, Resources, Archives. Agent hiểu được cấu trúc công việc, biết gì đang active.
2. **Infrastructure Layer (MCPs & CLIs)** — Authenticated access vào internal tools: docs, messaging, task trackers, code reviews
3. **Agent (Claude Code + Harness)** — Execution environment: filesystem access, tool calling, MCP integration
4. **Skills (Markdown + Scripts)** — Reusable workflows encoded as plain markdown

## Progressive Disclosure

- Root CLAUDE.md (identity + active portfolio)
- Project CLAUDE.md on demand
- Giải quyết context window limits — không load tất cả mọi thứ

## Key Learnings

1. Infrastructure comes first — authenticated access separates useful agent from chatbot
2. Progressive disclosure outperforms context dumping
3. Low-friction onboarding drives viral adoption
4. Your users are your best builders
5. Composability creates more value than features
