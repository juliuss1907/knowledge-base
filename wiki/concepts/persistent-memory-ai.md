# Persistent Memory AI

## Definition

Khả năng của AI agent duy trì thông tin qua các phiên làm việc — nhớ context, preferences, và kết quả từ các task trước để cải thiện performance lần sau.

## Key ideas

- **Self-learning loop**: Tự động adjust behavior dựa trên feedback và usage patterns
- **Giảm thời gian task lần sau**: Không cần re-explain context mỗi lần
- **Tự động setup skills**: Thêm skills mới khi thấy cần thiết
- **Khác biệt với Claude**: Claude không có persistent memory tương tự
- **Phù hợp recurring tasks**: Operator role yêu cầu memory để cải thiện theo thời gian
