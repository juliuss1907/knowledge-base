```markdown
---
[frontmatter]
---

# Article Title

## Metadata
[...]

## Summary

This article discusses RAG pipelines.    ← Only 1 sentence

## Key points
[...]
```

**Issue from report:**
```
**File:** wiki/sources/src_short-summary.md
**Severity:** WARNING
**Issue:** Summary is only 1 sentence, should be 3-5
**Suggested fix:** Expand summary using LLM
```

### After fix

```markdown
---
[frontmatter]
---

# Article Title

## Metadata
[...]

## Summary

This article discusses RAG (Retrieval-Augmented Generation) pipelines and their implementation in production systems. The author explains how RAG combines semantic search with LLM generation to provide more accurate and contextual responses. Key considerations include chunking strategies, embedding models, and retrieval optimization techniques.

## Key points
[...]
```

**LLM prompt used:**
```
Expand this summary to 3-5 sentences while keeping the same language:

This article discusses RAG pipelines.

Full content for context:
[article body]

Output only the expanded summary, no explanation.
```
