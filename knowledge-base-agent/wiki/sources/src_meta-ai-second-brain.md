# How We Built an AI Second Brain for 60K Knowledge Workers (Meta)

**Author:** Analytics at Meta  
**Link:** https://medium.com/@AnalyticsAtMeta/how-we-built-an-ai-second-brain-for-60k-knowledge-workers-78c507dd795b

---

## Mô tả

Meta xây dựng **AI Second Brain** — từ experiment trong analytics org đến **63,000+ users** across mọi function chỉ trong 3 tháng.

---

## 4 Components chính

1. **PARA Workspace** — Projects, Areas, Resources, Archives
2. **Infrastructure Layer (MCPs & CLIs)** — Authenticated access vào internal tools
3. **Agent (Claude Code + Harness)** — Execution environment với tool calling
4. **Skills (Markdown + Scripts)** — Reusable workflows encoded as plain markdown

---

## Key Learnings

1. Infrastructure comes first
2. Progressive disclosure outperforms context dumping
3. Low-friction onboarding drives viral adoption
4. Your users are your best builders
5. Composability creates more value than features
