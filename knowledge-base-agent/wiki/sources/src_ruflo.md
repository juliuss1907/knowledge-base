# Ruflo — Agent Orchestration Platform for Claude

Owner: ruvnet
Repo: ruflo
Link: https://github.com/ruvnet/ruflo

## Ghi chú chính
- **Mục đích**: Nền tảng điều phối (orchestration) agent chuyên sâu cho Claude, cho phép triển khai các "đàn" agent (multi-agent swarms) và các luồng công việc tự trị (autonomous workflows).
- **Tính năng cốt lõi**:
    - **Agent Swarms**: Điều phối nhiều agent làm việc cùng nhau.
    - **Self-learning Swarm Intelligence**: Hệ thống tự học và cải thiện khả năng điều phối.
    - **Tích hợp sâu**: Hỗ trợ native cho Claude Code, Codex và MCP (Model Context Protocol).
    - **Hệ sinh thái**: Có 32 plugins hỗ trợ mở rộng.
- **Điểm mới cập nhật**:
    - Tích hợp ruvllm và graph-node intelligence backends.
    - SONA native routing với hệ thống fallback 5 tầng.
    - Sử dụng Real Product Quantization với k-means.
    - Quản lý agent, causal edges và swarm teams thông qua graph-node records.
- **Quy mô**: Dự án rất lớn với 42.8k stars và hơn 6k commits.
