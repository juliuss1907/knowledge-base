# Personal Analyst Workflow

## Definition

Workflow sử dụng Hermes như personal analyst cho nhiều vai trò: research analyst, investment analyst, trader analyst, sales agent, customer support — với persistent memory và self-learning capabilities.

## Key ideas

**Use cases:**
- **Research analyst**: ingest X, Reddit, IG, Tiktok, formulate opinions
- **Investment analyst**: knows holdings, portcos, thesis → identify undervalued opportunities
- **Trader analyst**: execute trades, learn from success/mistakes
- **Sales agent, customer support, ops, accountant, finance ops**

**Workflow mới:**
- **Cũ**: scour X → Rabby → Coingecko
- **Mới**: mở Discord → check Hermes → provide feedback → improve

**Training focus:**
1. Track macro + geopolitics
2. X bookmarks summarize
3. Portfolio report
4. Learn from smart people

**Cost:** $5-10/tháng sau optimization
