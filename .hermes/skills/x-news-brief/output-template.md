# Output Template — X News Brief

Standard template for all crypto + tech briefings.

---

## Template Structure

```
✖️ X News Brief [☀️ Morning / 🌤 Midday / 🌙 Wrap] — [Day], [Date]

🔥 Top 2-5 stories
• [Story 1 title] — [1-line summary] ([views])
• [Story 2 title] — [1-line summary] ([views])

🧠 Narrative pulse
Crypto: [1-2 sentences về crypto trends]
Tech: [1-2 sentences về tech trends]

📌 Đáng đọc (7-10 items)

Crypto:
• [Item 1] — [Why interesting] [Link]
• [Item 2] — [Why interesting] [Link]

Tech:
• [Item 1] — [Why interesting] [Link]
• [Item 2] — [Why interesting] [Link]

TL;DR: [1 dòng tóm tắt covering both categories]
```

## Session Icons & Names

| Session | Icon | Time (ICT) | Coverage |
|---|---|---|---|
| Morning | ☀️ | 7:00 | Overnight + Asia morning |
| Midday | 🌤️ | 13:00 | US morning news |
| Wrap | 🌙 | 21:00 | US afternoon + day recap |

## Section Guidelines

### 🔥 Top Stories (2-5 items)

**Purpose:** Highlight most important developments (breaking + high engagement)

**Selection criteria:** All breaking (Tier 1) + highest engagement (Tier 2). Flexible mix, not forced 50/50.

**Format:** `• [Author/Source]: [Summary] — [Additional context] ([views])`

**Length:** 1 line per story, max 280 characters

### 🧠 Narrative Pulse (2 sections)

**Purpose:** Synthesize broader trends in each category

**Crypto (1-2 sentences):** Dominant theme, emerging trends, sentiment shift
**Tech (1-2 sentences):** Dominant theme, product launches, industry movement

**Tone:** Informative, neutral, no speculation

### 📌 Đáng Đọc (7-10 items, 50/50 split)

**Format:** `• [Title/Summary] — [Why interesting] [Link]`

**"Why interesting" patterns:** Breaking news, High engagement (X%), Viral momentum, Emerging topic, Unexpected take, Major launch, Developer tool

**Length:** 1 line per item, max 200 characters

### TL;DR (1 line)

**Format:** `[X] breaking, [Y] crypto + [Z] tech updates. [Crypto theme]. [Tech theme].`
**Length:** Max 280 characters

## Formatting Rules

- **Links:** Markdown `[Link](https://...)`
- **Numbers:** Views as "1.2M"/"500K", percentages 1 decimal
- **Emoji:** Section markers only (🔥🧠📌)
- **Line Breaks:** 1 blank between sections, none within

## CT Voice Guidelines

- **Tone:** Casual, informative, no hype
- **Slang:** Max 1-2 terms per brief (ser, alpha, based). Avoid wagmi/LFG/degen for news.
- **Recommendation:** 0-1 slang per brief for news briefing

## Edge Cases

### No new items → Simple message with last timestamp
### One category unavailable → Warning + continue with available data
### Extreme breaking (>5 items) → Expand top stories to 6-7

## Quality Checklist

**Content:** [ ] All items newer than last run [ ] Breaking included [ ] Top stories 2-5 [ ] Đáng đọc 7-10 (50/50) [ ] Narrative separate [ ] TL;DR covers both

**Format:** [ ] Session icon correct [ ] Date/day correct [ ] Links working [ ] Numbers consistent

**Tone:** [ ] Informative not hype [ ] Minimal slang [ ] No speculation
